Geoscience INTEGRATOR module

Geoscience ANALYST is the 3D front end to connect to, query, and visualize data from Geoscience INTEGRATOR, the software solution by Mira Geoscience for 4D multi-disciplinary exploration and geotechnical data management system.

Geoscience INTEGRATOR query

To connect to the server:

  1. In the Geoscience INTEGRATOR query window, click the Connect now button.
  2. In the credentials dialogue, enter your email, password, and the URL address of a valid Geoscience INTEGRATOR server.
  3. Click OK.
  4. In the drop-down menu of the Projects section of the credentials dialogue, locate the desired Geoscience INTEGRATOR project and click LOAD.

Once connected to a Geoscience INTEGRATOR project, you will see the project summary information displayed in the data table in the workspace tab (below the viewport). This includes the project’s name and description, the XYZ extents of the data, and a list of the the files uploaded via the Geoscience INTEGRATOR web browser user interface.



Viewing data sets/projects in the Object tree

After connecting to Geoscience INTEGRATOR and loading a project, all of its data will appear in the Geoscience ANALYST bjects panel. The projects are organized at the theme/sub-theme/object level.

 

Note: Greyed out data set names are data set placeholders.

 

Geoscience INTEGRATOR query parameters

The Query parameters appear in the Geoscience INTEGRATOR Query panel, at the right of the Geoscience ANALYST user interface. These are a series of filters as well as a tool to import certain objects and information. You can toggle on and off each one using the check box next to a parameter's name. Once selected, clicking Run at the bottom of the panel will filter and import your selections.

Bounding box

The bounding box query will filter the data based on X, Y and Z limits. By default, the from and to parameters are set to the extents of the entire data set. This option is available on all data themes.

To modify the bounding box extents:

  1. Digitize new from and to locations by either clicking on the corresponding target icon to the right, entering new values in the text boxes, or using the up and down arrows.
  2. Click Reset to max to revert to the entire X, Y, and Z extents.

To generate a bounding box that covers the area where most data is concentrated, click 2 standard deviations.

You can visualize the bounding box of this area of concentration clicking Show selection bounding box the icon to the right.

You can drag a line in the viewport to nominate the X, Y, and Z extents. We recommend setting a view from top of the bounding box to draw the X and Y extents, and from a side view when drawing the Z extents.

Date

This option filters time-based data of the event-type themes. You can select dates using the calendars or enter them in the text boxes.

Another version of this filter is available to query the themes containing objects with time-related data. Since these themes can hold many instant data for each object location and Geoscience ANALYST can only show one property value at a time per location, you need to provide a datetime value for it to select the instant properties as of a specific datetime. The system will return the values that correspond exactly to the specified datetime, or the most recent one before.

Note: If you are a Geoscience INTEGRATOR user, note that using this query in the Underground theme functions like the Compiled table query, as opposed to the data set explorer query.

Object (name of object type varies from theme to theme)

This option refers to samples and other specific types of objects that you can use as filters in the data set explorer.

Property Filters

This option filters by property values. Press + to add another property filter.

Note: The nature of the data in a theme will determine the query behaviour and options available in the property filter.
  • Text data: Is is the only option available. You must enter the entire character string in the text box.
  • Classification and binary: Is is the only option available. Expand using the down arrow and select one or more classes to import.
  • Float, integer, and date: Select one of the available operations and enter a value in the text box.

For Shifts, Properties, Tags, and Groups. simply toggle on using the checkbox and select among the available items in the project to import them. These options only appear if they are present in the working data set.

The Properties section lists all available properties in a data set, but note that time interval data is not supported and will appear greyed out in the Geoscience INTEGRATOR Query\Properties panel.

When you have set all the desired filters, you can click Run at the bottom of the Query panel to send the query. Geoscience ANALYST will only load the pertinent records, as per query results.


Viewing data import results

Geoscience ANALYST stores the data import results on special object types, depending on the data type. For example, microseismic events are a special points object that stores the data on nodes referred to as events.

For data sets that have groups (e.g. Earth models/ > 3D point sets), the system stores the data imported on objects named according to the group, located in a timestamped folder within the data set group. Similarly, drillholes are objects located within a timestamped folder (highlighted in image below), within the drillholes data set group. All other queries will be objects named according to a timestamp directly under the data set folder.

The viewport displays the imported data in 3D, and the data table displays a summary.

 

Viewing project files, maps, and documents

You can find project files, maps, and documents in the Workspace tab of the data table, where you can use a search bar to locate files of interest. Clicking on a file name will display a selection of options:

Import: Imports the file under the project or data set name in the objects panel. The data file has to be in a format supported by Geoscience ANALYST, for example GOCAD surface files, DWG and DXF files.

Open: The system will try to open the file using the program associated to that file type on the computer. For example, a photo file saved with the extension .JPG will open in an image viewer application.

Save as: Downloads the file to your computer.

Pull: Downloads the file in the Geoscience ANALYST workspace and becomes available offline.

Cancel: Cancels the file selection.

 

 

Visualizing the data in 3D

Once you have imported your data into Geoscience ANALYST, you can take advantage of the numerous visualization tools to analyze it. You can view integrated data and models from multiple data sets and files together and analyze, interpret and share with others.

Before exiting Geoscience ANALYST, you can save the imported data using the File menu option. Doing so will create a Geoscience ANALYST file on your computer, which will remain available even after terminating the connection to Geoscience INTEGRATOR.

Note: It is only possible to connect to one Geoscience INTEGRATOR project at a time.
