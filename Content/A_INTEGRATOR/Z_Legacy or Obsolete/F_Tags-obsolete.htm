Tags

Tags are alphanumeric identifiers you can assign to data in Geoscience INTEGRATOR, giving you the ability to associate and filter data across various themes within a project.

There are three types of tags in Geoscience INTEGRATOR:

Tag type Description
Project Tags assigned by any user who has imported data into a project.
User Tags assigned only by the user who created the tag.
System theme-specific tags automatically assigned at the system level.

 

Note: The ability to create a tag depends on user permissions defined by the system administrator. Under default settings, all users can view tags but only system administrators and project managers can create tags.

To create a User or Project tag:

Tags.

In the Current tags table, click the Add new tag button.

In the Add new tag dialog box, give your tag a name and a description (optional).

Define the tag type by selecting Project or User.

Click Add, and then click Close.
