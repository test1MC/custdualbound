lProjects and user roles

The term project refers to the upper-most sub-division of data, which could be any of the following:

  • single mine site
  • group of mine sites sharing data
  • resource development project
  • exploration project.
  • multi-disciplinary research project
  • study containing multiple components of Geoscience-related data
  • feasibility project

 

 

Users and roles are as follows:

User Role
System administrator
  • Highest user level
  • Can access every feature of Geoscience INTEGRATOR
  • Builds list of permissions for each user, to determine who can create, edit, upload, export, view data
Project manager
  • Can access every feature of Geoscience INTEGRATOR
  • Can create, edit, upload, export, view data
Power user  
Limited user
  • Can view data and reports uploaded to projects
  • No editing permission
