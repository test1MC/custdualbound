Creating a new project

In most cases, it will be the task of the system administrator to create a new project in Geoscience INTEGRATOR. In some cases, this task may fall on the project manager.

A project must contain data that either the system administrator or the project manager must upload to Geoscience INTEGRATOR.

Once you have gathered the data needed for a project, the following instructions will guide you through the process of creating a new project.

To create a new project:

  1. In the Geoscience INTEGRATOR menu, click System administration(i), then click Projects(ii).
  2. At the top right of the Projects table, click

  3. In the Add new project dialog box, do the following
    1. Give the project a name and a description (optional);
    2. Define the units used for the XYZ coordinate system e.g., m for metres;
    3. Click Add;
    4. Once you see the Project successfully uploaded message click Done.
  4. In the Current project drop-down list at the top of the screen, navigate to your newly-created project and click on it.
