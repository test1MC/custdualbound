Data fusion & samples

In the menu under Data fusion, the Details tab in Samples compilation contains a list of the each sample and its unique ID within your project. This table allows you to view and search sample metadata.

 

The Chart | Table tab opens 's powerful filter, chart, and table tools, giving you the ability to filter and view your data in numerous precise presentations and combinations.

 

In , there are three themes that make up the Samples family. They are as follows:

Samples family Geochemistry & minerology
Samples
Physical properties
Drillhole intervals

 

A sample record is made up of one or many properties from themes in the Samples family. For example, you may upload both physical property data and geochemistry data for the same rock sample.

 

Note: Although the filter, chart, and table tools in both the data set explorers and the Data fusion & samples appear and function alike, data set explorers' data filtering and presentation is theme-specific. The key difference with the Samples filter, chart, and table tools is that they filter and present data simultaneously from multiple properties and themes throughout the Samples family.
