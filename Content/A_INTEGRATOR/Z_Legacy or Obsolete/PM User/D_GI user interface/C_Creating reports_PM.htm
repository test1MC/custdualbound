Creating reports

allows you to create reports that will help you to document and monitor issues within projects.

Creating a report and adding content to it from your project take place in two separate modules of the user interface. The two sets of instructions below illustrate how.

To create a report:

  1. In menu, click Reports (i), and then the Add new report button (ii).
  2. In the Reports template, complete the following fields:
    1. Name and description
    2. Status, scheduling, and frequency (optional)
    3. Add attachment (optional)
    4. Tag users by email (optional)

  3. Click Save.

 

To add items to an existing report:

  1. Navigate to the location of the data you wish to add to your report. This could be:
    1. The View data tables or View data charts interfaces of the Reporting module
    2. The tables and/or charts interfaces of the Sample compilation module
    3. The data set explorers.
  2. Filter your data to create a chart and/or a table.
  3. Click Apply changes.
  4. Click Add report item.
  5. In the Add a new report item dialog box, select the report to add the item to, and name the report.
  6. Click Add, then click Close.

You can now view, edit, delete, and/or send your report with the added item(s) by navigating to the Existing scheduled reports table in the Reporting module.
