Dashboards

Geoscience INTEGRATOR’s dashboards are a quick and easy way to review information related to a project and its data sets. This section provides information on the following key topics and operations:

  • Project dashboard
  • data set explorers
  • Properties
  • Filtering data
  • Charts
  • Tables

Project dashboard: The full view

The Project dashboard appears when you log in to Geoscience INTEGRATOR, displaying the Current project table and the All projects table. These tables display general information pertaining to the projects, including the project name and description, as well as basic user, date, and modification information.

Click the themes with data column to display a dynamic list of a project’s theme(s) and any data associated with the theme(s). Selecting a theme in the list opens its data set explorer.

 

Note: The Active project open prior to closing Geoscience INTEGRATOR will relaunch when opening the program the next time.

 

data set explorers: Making the most of your data

At the heart of Geoscience INTEGRATOR lies the data set explorer. Here you have access to numerous resources and tools designed to help you analyze your data at the granular level, and in as many ways as possible.

With your templates created and data imported, you will find your data sets available in the data set explorer.

To open the data set explorer:

  1. Click the Current project list on the selection bar, and select the project you plan to work in from the drop-down list.

  2. In the Geoscience INTEGRATOR menu, click data set explorer.
  3. In the table under Choose a data set, locate the data set you wish to view.

There may be one or several data sets in the dashboard, depending on the scope of the project. The table provides a summary of each data set, which displays the statistics related to the mandatory and user-created properties defined in the import templates.

The data set summaries include several fields that display depending on if a specific attribute is set in a project template. Geoscience INTEGRATOR refers to these attributes as Project settings. When a Definition has data, it will be available to view in the data set summary by clicking on it, as shown in the image below represented by blue hyperlinked text. Therefore, data set summaries do not always contain the same exact Project settings.


 

Properties

Properties are attributes of data such as XYZ coordinates, sensor IDs, intervals, etc. In Geoscience INTEGRATOR, properties can take alphanumeric, binary, classification, date-time, float, or integer form.

There are two types of properties in Geoscience INTEGRATOR: mandatory properties and user-created properties. Mandatory properties are essential for a data set, and the properties themselves depend on the nature of the theme that they belong to. For example, mandatory properties for a data set of drillholes would be:

  • drillhole ID
  • x (metres)
  • y (metres)
  • z (metres)
  • length

 

Note: If a mandatory property is not present in the original file you will be importing, an error message will appear and the file will not upload.

User-created properties are non-essential and are more flexible. Users can add any number of properties to a data import template and customize them according to their own systems of classification. For example, with Earth model points data, it is mandatory to provide X, Y, and Z coordinates, while user-created properties may include any other variables such as rock type, depth, and distance to faults.

 

Filtering your data sets

The data set explorer contains a robust filtering tool to help you analyze data at the granular level. There are many ways to customize the way you want to view and present your data through the numerous chart and table options.

You will find the filtering tool in the toolbar, shown at right.

The filtering tool allows you to sort, select, and multi-select attributes of your data, as illustrated in the table below:

 

   
Neighbourhoods Spatial regions already defined in your data set.
Shifts Time-based properties that help investigate events occurring between defined times of the day and night.
Tags User-created alphanumeric identifiers associated with the data that help to identify, sort, and locate specific excerpts of data.
Files

Any media uploaded to the system relevant to a data set acting as supplementary information.
Property Filters Filters numeric data using the operators is, is not, >=, or <=. Filters properties with alphanumeric data using the operators is, or is not.
Samples A catalogue of the project where you check one, many or all the samples contained within.
Sample media The actual element that makes up the sample, e.g. gas, rock, soil, water, etc.
Under detection limit

The lowest quantity of a sample that can be reliably detected through reliable analysis.

There are two options to filter by, based on fields created in the import template:

Display format specifies what string of characters to look for in the CSV file

Display valuespecifies what numerical value to put in the database in its place, to use for calculations.

These results will display in the tables.

 

Note: The filtering tool displays drop-down menus that correspond directly with data defined within a data set. Therefore, the filtering options vary between data sets, depending on the data provided.

To filter your data:

  1. In the Geoscience INTEGRATOR menu, click data set explorer.
  2. Select the data set you wish to view.
Note: When a theme in a project has only one data set, the filtering tool window will automatically display at right upon opening the data set summaries table. If there is more than one data set, you must click one of the data sets for the filtering tool window to open.
  1. Decide which variables/attributes/Project settings you wish to filter, and apply them to your filter by navigating the drop-down menus available in the filtering tool.
  2. Click Apply changes in the toolbar at right.

 

Charts

You can view your filter results by through numerous chart options, each containing multiple properties and settings to help you fine-tune the details of your data and how you wish to present it. You will find the Charts button in the toolbar, at right.

The following chart types are available for all properties:


Scatter plots Histogram
   
Ternary Plots Horizontal bars

The following chart types are available for time-based properties:

  • Time series
  • Time histograms
  • Time scatter plots
  • Numeric plots

Choosing a chart type

By default, selects the first property of your data for the x axis, and the second for the y. However, you can select properties for the x and y axes. Histogram and Horizontal bars charts only contain x and y axis parameters, while Scatter, Time scatter, and Ternary plots allow you to add and view additional dimensions.

To access these features, click the Other button in the Charts menu. They are as follows:

Other chart properties Description
Radius Adjusts the size of the points on the chart based on a numeric property additional to the x and y axes.
Radius value Adjusts the size of points where highest value equals highest or lowest radius.
color Assigns colors to the points based on either a numeric property or a classification.
color spectrum Assigns colors defined by either red (min) to blue (max), or blue (min) to red (max) for numeric properties. If using a property defined as a classification, you can define colors in the Classifications menu.
Tooltip properties

Queries individual points within a chart, and returns one or several properties associated with that point. Selecting the properties of interest and hovering the cursor over an individual point displays an information box presenting the values of the property associated with the point.

 

To view your chart, click the Apply changes button in the toolbar.

Tables

You can customize a table by selecting and deselecting properties, ordering the columns, sorting data ascending or descending, and selecting the number of rows to display. You will find the Tables button in the toolbar, at right.

You can export the table data in CSV format by clicking Export to CSV at the top right of the table.

To view your table, click the Apply changes button in the toolbar.
