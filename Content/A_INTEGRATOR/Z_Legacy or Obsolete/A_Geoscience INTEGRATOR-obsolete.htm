About

Geoscience INTEGRATOR is a multi-user, multi-site data management system for geotechnical, mining, and exploration data, which can be customised for the specific data management needs of individual mine sites. It excels as a tool to store, group, and query multi-disciplinary exploration, mining, and geotechnical data, models, files, and documents. It has powerful and intuitive query capabilities that can generate and manage scheduled, data-triggered, or on-demand reports. It is the foundation to bring many types of different data together for integration and modelling.

As a true 4D data management system (x, y, z, time), Geoscience INTEGRATOR natively handles 3D spatial and temporal data for any data type for which spatial position or time is relevant. The diagram below illustrates the general system configuration from a user perspective.

 

The data management system is server-based, with client applications accessing the server over a private network or the Internet. Users gain access through a fine-grained permissions system, under the control of the system administrator. Two client applications are provided:

  1. A web browser for system setup and management; data import and export; display of files, documents, tables, and charts of data; and customised report set-up through either self-serve or system-defined reporting, accessed directly in the web browser or by automated email distribution.
  2. A 3D visualisation application that displays data in the context of maps and 3D geological models. Both the web browser and 3D visualisation applications support complex spatial, temporal, and property-value queries. Additional applications can be developed by end users using Geoscience INTEGRATOR’s Application Programming Interface (API) provided with the system.

The system connects to site data sources for real-time, continuous update or ad-hoc update. Data entry to the system is simple and can be automated according to user-customized specifications. Data streams are captured, quality-controlled, and conditioned by the system directly from web browser entry, from other site databases, or from flat files on a file network system. The management of documents, files, and their links to data is simple and effective.

 

Features   Analysis and reporting
  • Provides a user friendly system to query and analyze 4D data.
  • Facilitates knowledge transfer amongst mine personnel.
  • Focuses users on interpretation and analysis, not on data handling.
  • Adds value to systems already in place by increasing their efficiency and scope without removing any important outputs.
  • Ensures data integrity in a secure, managed environment.
  • Provides a stable archive of the mine’s geotechnical and production data which is captured for long-term archive, query, and retrieval.
  
  • Direct web browser access to data and reports.
  • Comprehensive chart and tabulation options for data analysis.
  • Innovative tags, shifts, and neighbourhood queries to personalize and interrogate data.
  • Custom designed summary reports dispatched by email to designated recipients.
  • Scheduled, data-triggered or on-demand reports.
  • Powerful 3D visualization environment with a query interface that allows the selection of data by type, data set, time range, spatial location, value ranges, and other query parameters.

 

Data input

Geoscience INTEGRATOR harnesses the powerful capabilities of on-site databases that perform complex functions, using the results to compute and deliver intricate and highly meaningful data representations.
