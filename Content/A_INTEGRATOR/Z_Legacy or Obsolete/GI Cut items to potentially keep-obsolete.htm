When the following filtering tools contain records, they will be available regardless of the selected theme:

The following filtering tools are available depending on the selected theme:

  • Sample: A list of all the sample IDs in the selected data set where you check one, many, or all the objects contained within the project. (Samples, Geochemistry, Physical properties)
  • Sample media: The actual element that makes up the sample, e.g. gas, rock, soil, water, etc. (Samples, Geochemistry, Physical properties)
  • Under detection limit: This gives you the choice of how you wish to display datathat is under the detection limit in the table view. Because UDL is indicated in text format and data is transposed in a numerical format in the database, you can choose to display it in the text format, or in the numeric format.
