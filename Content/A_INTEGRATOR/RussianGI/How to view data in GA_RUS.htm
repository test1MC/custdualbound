Трехмерное отображение данных и анализ

Программа Geoscience ANALYST обеспечивает мощную среду трёхмерной визуализации с интерфейсом построения запросов, которая позволяет анализировать данные и файлы системы Geoscience INTEGRATOR в пространственном контексте.

После установки программы Geoscience ANALYST на компьютере, она подключается к системе Geoscience INTEGRATOR через частную сеть или сеть Интернет.

Просмотр данных системы Geoscience INTEGRATOR программе Geoscience ANALYST:

  1. Запустите программу Geoscience ANALYST и:
    1. Нажмите вкладку Запрос в системе Geoscience INTEGRATOR, затем
    2. Нажмите кнопку Подключиться. Также можно выполнить подключение при помощи пункта Подключиться к Geoscience INTEGRATOR в меню «Файл». (Этот метод позволяет выполнить подключение к другому проекту без необходимости перезагружать приложение.)



  2. В диалоговом окне Учетные данные для программы Geoscience Integrator:

    1. Введите ваш адрес электронной почты, пароль и URL-адрес действующего сервера Geoscience INTEGRATOR;
    2. Нажмите OK.
    3. В раскрывающемся списке выделите проект, затем
    4. Нажмите ЗАГРУЗИТЬ.

После подключения к проекту Geoscience INTEGRATOR вы увидите информационную сводку по проекту, отображаемую в таблице данных на вкладке «Рабочая область» (внизу окна просмотра). Эта информация будет включать в себя имя и описание проекта, границы проекта в координатах XYZ, а также список файлов, загруженных через веб-интерфейс системы Geoscience INTEGRATOR

С левой стороны пользовательского интерфейса Geoscience ANALYST вы увидите, что в панели «Объекты» содержатся темы, наборы данных которых находятся в данном проекте. При развертывании этих вкладок отрывается список наборов данных в каждой теме.

Примечание: Серым цветом затенены имена наборов данных, которые являются заполнителями пустых наборов данных.

При нажатии на имя набора данных, в этом списке загружается суммарная информация набора данных и отображается в Таблице данных. Несмотря на иное форматирование, эта информация соответствует информации, которая представлена в Диспетчере базы данных пользовательского интерфейса Geoscience INTEGRATOR, включая список файлов, привязанных к набору данных. На панели запросов в системе Geoscience INTEGRATOR отображены специальные инструменты фильтров для набора данных, а красный прямоугольник в окне просмотра отображает границы данных.

Как и в Диспетчере базы данных, инструменты построения выборки - фильтры здесь индивидуально подобраны в зависимости от темы. Выбрать необходимые фильтры можно, включая или отключая соответствующие флажки.

Параметры запроса

Параметры запроса появляются в панели Запроса в системе Geoscience INTEGRATOR в правой стороне пользовательского интерфейса Geoscience ANALYST . Здесь находится несколько фильтров, а также инструмент импорта определенных объектов и информации. Каждый их элементов может быть включен или отключен при помощи флажка, расположенного рядом с наименованием параметра. После выбора инструмента нажмите Выполнить внизу панели, чтобы применить фильтры и импортировать выбранные элементы.

Ограничивающий объём

Запрос при помощи ограничивающего объёма позволяет произвести выборку данных на основе границ X, Y и Z. По умолчанию, параметры от и до распространяются до границ всего набора данных. Эта опция доступна во всех темах данных.

Чтобы изменить значения:

  1. Выполните оцифровку местоположения точек от и до нажатием на соответствующую иконку мишени справа, вводом новых значений в текстовые поля или при помощи стрелок управления курсором.
  2. Нажмите Установить на максимум, чтобы отменить изменения всех границ X, Y, и Z.

Чтобы создать ограничивающий прямоугольник, охватывающий область, где сосредоточена большая часть данных, нажмите 2 стандартных отклонения.

Чтобы отобразить прямоугольник, ограничивающий зону сосредоточения данных, нажмите иконку справа.

Чтобы задать границы X, Y и Z, можно передвинуть линию в камере. Рекомендуется устанавливать вид сверху, чтобы прорисовать границы по координатам X и Y, и устанавливать вид сбоку для прорисовки границ Z.

Дата

Эта опция позволяет применять фильтры к данным с временным критерием, например микросейсмическим данным. Даты можно вводить при помощи календарей или вводить в текстовых полях.

Ещё один вариант этого фильтра доступен для создания выборки в темах по объектам с данными с временным критерием. Поскольку в таких темах может содержаться множество моментальных данных для местоположения каждого объекта и программа Geoscience ANALYST может отобразить одновременно только одно значение свойства для одного местоположения, необходимо ввести для него значение даты-времени, чтобы выбрать моментальные и интервальные свойства соответствующей даты и времени. Система выдаст значение ближайшей моментальной записи перед указанной датой и запись интервальной даты, которая включает в себя эту дату (если таковая имеется).

Объект (имя типа объекта в различных темах может отличаться)

Эта опция относится к пробам и прочим специальным типам объектов, которые могут быть использованы в качестве фильтров в Диспетчере базы данных.

Фильтры свойств

Эта опция позволяет сделать выборку по значениям свойства. Нажмите +, чтобы добавить фильтр по свойству.

Примечание: Режим выполнения запроса и опции, доступные в фильтре свойств зависят от специфики данных в отдельных темах.

 

 

Скважины

Эта опция относится к пробам и прочим специальным типам объектов, которые могут быть использованы в качестве фильтров в Средстве анализа набора данных.

Фильтры свойств

эта опция позволяет сделать выборку по значениям свойства. Нажмите + , чтобы добавить фильтр по свойству.

Примечание: Режим выполнения запроса и опции, доступные в фильтре свойств зависят от специфики данных в отдельных темах.
  • Текстовые данные: Доступна только одна опция Есть. Необходимо ввести текстовую строку целиком в предоставленное текстовое поле.
  • Классификация и двоичные данные: Доступна только одна опция Есть. Разверните при помощи стрелки «вниз» и выберите один иди более классов для импорта данных.
  • Число с плавающей точкой, целое число, и дата: Выделите одну из доступных операций и введите значение в текстовом поле.

Для тем Смены, Свойства, Метки, и Группы просто поставьте флажок и выберите необходимые среди доступных элементов проекта, чтобы выполнить их импорт.

После установки всех необходимых фильтров нажмите Выполнить внизу панели «Запрос», чтобы отправить запрос. Программа Geoscience ANALYST загрузит только соответственные записи согласно результатам запроса.

 

Просмотр результатов импорта данных

Программа Geoscience ANALYST сохраняет результаты импорта данных в специальных типах объектов, в зависимости от типа данных. Например, микросейсмические события являются особым точечным объектом, где данные сохраняются в узлах, которые называются события.

Для наборов данных, которые формируют группы (например, модели геологической среды–трёхмерные наборы точек), система сохраняет импортированные данные на объектах, которым присваиваются имена в соответствии с группой, расположенной в папке с отметкой времени в группе набора данных. Аналогично, скважины являются объектами, расположенными в папке с отметкой времени (выделение на рисунке ниже) в группе наборов данных «скважины». Все остальные запросы будут объектами с именем в виде отметки времени непосредственно в папке набора данных.

В окне просмотра отображены импортированные данные в трёхмерном изображении, а в таблице данных показана итоговая сводка.

 

Просмотр файлов, карт и документов проекта

Документы, карты и файлы проекта расположены на вкладке Рабочая область таблицы данных, где при помощи строки поиска можно найти необходимые файлы. При нажатии на имя файла на экран будет выведен список опций:

Импорт: Импорт файла под именем проекта или набора данных в панели «Объекты». Файл данных должен быть в формате, поддерживаемом программой Geoscience ANALYST, например, файлы поверхностей GOCAD, файлы DWG и DXF.

Открыть: Система попытается открыть файл при помощи программы на компьютере, ассоциированной с таким типом файлов. Так, например, файл фотографического изображения, сохраненный с расширением .JPG, будет открываться в приложении типа средства просмотра изображений

Сохранить как: Сохранение файла на компьютер пользователя.

Скачивание: Загрузка файла в рабочее пространство Geoscience ANALYST, при этом файл становится доступен для автономного использования.

Отмена: Отмена выделения файла.



Отображение данных в трёхмерном представлении.

После импорта данных в программу Geoscience ANALYST становятся доступны преимущества различных средств визуализации. Пользователю предоставляется возможность просматривать интегрированные данные и модели, собранные из различных наборов данных и файлов одновременно, а также анализировать, интерпретировать данные и совместно использовать их с другими пользователями.

Перед выходом из программы Geoscience ANALYST, вы можете сохранить импортированные данные при помощи соответствующей опции в меню Файл. В процессе сохранения в компьютере будет создан файл Geoscience ANALYST, который будет доступен также после разрыва соединения с системой Geoscience INTEGRATOR.

Примечание: Одновременно может быть установлено соединение только с одним проектом Geoscience INTEGRATOR.
