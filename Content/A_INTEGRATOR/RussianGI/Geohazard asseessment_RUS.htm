Аналитика геотехнических рисков

Модуль аналитики геотехнических рисков поддерживает автоматический расчет оценки рисков, как на основе знаний, так и на основе значений данных. В модуле используются правила на основе различных пользовательских критериев. Обновление оценок рисков выполняется согласно заданному расписанию, или по команде пользователя. Сервер выполняет пространственные расчеты (например, близости к структурам), а также прочие четырёхмерные расчеты, относящиеся к рискам, (например, плотность микросейсмических проявлений) без участия пользователя.

Оценка рисков опасных геологических процессов выполняется в два этапа:

  • Объединение данных по точкам модели рудника с целью получения актуальных значений свойств.
  • Проверка выполнения правил возникновения геотехнических рисков с целью получения показателя геотехнического риска для точек модели рудника, а также расчета показателя достоверности.

 

Объединение данных: Сборка модели рудника

Для оценки рисков опасных геотехнических процессов в инфраструктуре рудника модуль Геотехнической аналитики использует точки наборов данных модели рудника в качестве основы для пространственного сближения. При помощи элемента Сборка модели рудника процесс объединения данных собирает все необходимые данные для расчета различных показателей геотехнических рисков.

На вкладке Обзор сборки модели рудника для каждого набора данных модели рудника в проекте отображается поле, в котором представлена суммарная информация по количеству точек, импортированным свойствам и свойствам, которые рассчитываются системой.

  • Импортированные свойства: Свойства, импортированные пользователем, которые не обновляются системой. Как правило, это фиксированные свойства, которые не будут изменяться со временем, например, дата отработки. Эти свойства, как правило, не нуждаются в обновлении перед проверкой выполнения правил определения геотехнических рисков.
  • Свойства, которые рассчитываются системой: Эти свойства являются результатом различных расчетов, выполненных с использованием других данных и/или файлов в проекте. Перед проверкой выполнения правил оценки геотехнических рисков важно обеспечить наличие актуальных значений переменных по времени свойств.
Примечание: Свойства, которые расчитываются системой можно активировать при помощи кнопки Рассчитать сейчас, или задать график выполнения расчётов.

Результаты обновлений свойств, рассчитанных системой, импортируются в набор данных модели рудника как моментальные данные с расчетом параметра дата-время как отметки времени. После актуализации свойств выполняется проверка выполнения правил геотехнических рисков.

 

Пульт управления геотехническими рисками

Пульт управления геотехническими рисками управляет расчетом показателей геотехнических рисков по требованию или согласно заданному расписанию.

Этот раздел пользовательского интерфейса Geoscience INTEGRATOR отображает информационную сводку по каждому геотехническому риску в проекте. В представленных полях отображены наиболее актуальные даты расчетов, даты следующих расчетов (если запланированы), а также имена ассоциированных наборов данных модели рудника. Также указывается состояние, которое может быть представлено следующими значениями: «готов», «в процессе расчета», «сбой» и «превышено пороговое значение».

Для того чтобы развернуть поле информационной сводки по геотехническим рискам, необходимо нажать мышью в любом месте поля. При этом будет отображена дополнительная информация, включая расписание актуализации показателей геотехнических рисков, пороговые значения и значения априорной вероятности, и полный перечень правил для расчета геотехнических рисков и показатели достоверности. При наведении указателя мыши на дату Последнего расчета (отображается оранжевым цветом) отобразится отметка времени для выполненного импорта для этого расчёта.

    Расчёты геотехнических рисков

    Геотехнические риски могут принимать различные формы. В каждом случае система производит оценку списка правил, соответствующих условиям в точках модели рудника по заданному набору данных.

    Эти правила основаны на задаваемых пользователем отношениях (на основе суммы знаний) или на анализе предшествующих корреляций (на основе данных). Каждое правило включает в себя одно или более свойств точек модели рудника и представляет собой определённое условие. Правила могут иметь положительный вес (W+) и отрицательный вес (W-).

    Если правило имеет только положительный вес, система применяет этот вес, если выполняется условие. Так например:

    Расстояние_до_контакта_руды < 15м W+ =10

    Для каждой точки набора данных модели рудника, если свойство с именем Расстояние_до_контакта_руды меньше 15, применяется положительный вес 10.

    Если правило имеет положительный и отрицательный вес, система применяет положительный вес, если выполняется условие, и отрицательный вес, если условие не выполняется. Так например:

    Расстояние_до_контакта_руды < 15м W+ =10 W- = -2

    Для каждой точки набора данных модели рудника, если свойство с именем Расстояние_до_контакта_руды меньше 15, система применяет вес 10, а если значение больше или равно 15, система применяет вес -2.

    Если оценка правила невозможна, система не применяет весов. В вышеприведенном примере, если для точки в наборе данных модели рудника значение Расстояние_до_контакта_руды отсутствует, для этого правила по данной точке вес не присваивается.

    После оценки всех правил для каждой точки в наборе данных модели рудника производится расчёт фактора опасности. Расчёт может быть простым и представлять собой сумму весов, или может быть более сложным при использовании правил экспертной системы на основе данных.

    Алгоритм весомости признаков

    Алгоритм весомости признаков - количественный вероятностный метод объединения данных в поддержку определенной гипотезы. Этот основанный на данных метод может применяться для оценки потенциальной будущей опасности на основе анализа предшествующих корреляций. При формировании правил для этого метода система использует следующие расчёты для получения коэффициента риска:

    Значение коэффициента риска

    Высокое значение коэффициента риска указывает на то, что большинство или все критерии, которые проверялись при помощи правил, были выполнены, и что все или большинство условий, которые могут привести к опасной ситуации, присутствуют.

    Чтобы моментально привлечь внимание к потенциально опасным зонам, пользовательский интерфейс четко отобразит точки, где показатель риска превышает пороговое значение. Детальная информация может быть получена в пользовательском интерфейсе в поле информации о геотехнических рисках и на странице Журнал расчетов, а также в автоматически генерируемом отчете, который рассылается по электронной почте.

    Показатель достоверности

    Система рассчитывает показатель достоверности для каждой точки. Показатель достоверности дает представление о достоверности рассчитанного показателя риска для каждой точки модели рудника.

    Система сохраняет показатели геотехнического риска с датой и времени проведения расчета в виде отметки времени в точках модели рудника как свойства моментальных данных.

    По завершении расчетов показателей геотехнических рисков система высылает один или более отчетов по электронной почте согласно заданному пользователем списку получателей. Существует два типа отчетов о геотехнических рисках: плановый отчет и экстренный отчет.

    Плановый отчет выдает подтверждение для инженеров-геотехников о том, что система выполнила расчеты, и пороговый показатель геотехнического риска не был достигнут ни в одной из точек выбранного набора данных модели рудника.

    Система высылает экстренный отчет только в тех случаях, когда показатель риска превышает заданное пороговое значение как минимум для одной точки модели рудника.

    Экстренный отчёт системы о геотехнических рисках содержит:

    • Имя геотехнического риска и дату-время расчета показателя риска.
    • Планы горизонтов для горизонтов, содержащих точки, в которых показатель риска выше порогового значения. На планах горизонтов точки модели рудника закрашены цветовым значением геотехнического риска.
    • Координаты xyz точки, где отмечено наивысшее значение показателя, а также перечень правил, условий и значений свойств, которые привели к повышенному показателю риска.

    Отчеты о геотехнических рисках, как и плановые отчеты, остаются в системе доступными для загрузки в течение одного месяца на странице Составление отчётов /Журнал отчетов.

    Заявление об ограничении ответственности

    Оценка геотехнического риска, которая предоставляется системой Geoscience INTEGRATOR, подпадает под действие правовых оговорок и положений об ограничении ответственности согласно договорам лицензии и предоставления обслуживания. Данная оценка геотехнического риска обеспечивает основу исключительно для общей оценки потенциального риска в будущем на основе задаваемых пользователем отношений или анализа предшествующих корреляций, поэтому допускают неточности. Результаты не являются прогнозом появления или непоявления определенных событий проявления геотехнических рисков. Все выходные данные должны проходить независимую проверку.
