Диспетчер файлов

Диспетчер файлов представляет собой хранилище информации в среде Geoscience INTEGRATOR, которое позволяет загружать, хранить, выбирать при помощи фильтров поиска и сортировать файлы различных типов, которые относятся к проекту. Менеджеры проектов и системные администраторы могут изменять список типов файлов на странице Типы файлов в меню Параметры проекта на левой панели. Файл может быть ассоциирован с метками и/или наборами данных.

Примечание:Файл может быть ассоциирован с отдельной записью данных; однако эта операция выполняется не в Диспетчере файлов, для этого предназначена страница Составление отчётов.


  • При нажатии на Диспетчер файлов (i) в меню открывается инструмент фильтра для файлов и сводная таблица информации о файлах.

  • Модуль Применить фильтр к файлам (ii) позволяет выполнить выборку файлов по имени, типу файла, диапазону дат, и авторам.

  • При нажатии указателем мыши на имени файла (iii) открывается специальная таблица, в которой перечислены наборы данных, связанные с этим файлом.

  • Кнопка +Загрузить новый файл (iv) открывает модуль внизу сводной таблицы данных, в котором предусмотрена форма для загрузки нового файла.

  • При нажатии (v) происходит удаление файла (в зависимости от уровня доступа), а при нажатии (vi) выполняется скачивание файла на жесткий диск вашего компьютера.

  • При нажатии на число в столбце Метки (vii) открывается диалоговое окно, в котором перечислены метки, связанные с файлом. В этом окне можно ассоциировать метки с файлами и отменять ассоциации.

  • При нажатии на число в столбце Наборы данных (viii) открывается диалоговое окно, в котором перечислены Наборы данных, связанные с файлом. В этом окне пользователи могут ассоциировать Наборы данных с файлам и отменять ассоциации.

Загрузка файлов

При нажатии кнопки +Загрузить новый файл открывается модуль, в котором предусмотрена форма для привязки метаданных, меток и наборов данных к файлу, который будет загружен.

При выборе меток и/или наборов данных в соответствующих списках будет выполнена их привязка к загружаемому файлу. Вы можете выбрать все элементы списка, а если список слишком велик, выполнить поиск необходимых элементов в списке. Метки и наборы данных в списке соответствуют меткам и наборам данных, доступным в проекте.
