Применение фильтров к данным и построение запросов

Диспетчер базы данных включает в себя надежный и очень гибкий инструмент фильтрации, который применяется при анализе подмножеств данных. Система предусматривает немало способов просмотра и представления данных при помощи многочисленных опций построения диаграмм и таблиц.

Инструмент фильтрации находится на панели инструментов, показанной справа.

После выбора одного из инструментов (фильтров, графиков, таблиц) пользователь видит серию раскрывающихся меню, которые прямо соответствуют данным, прописанным в наборе данных. Следовательно, настройки фильтрации в различных наборах данных отличаются друг от друга.

:Применение фильтров к данным:

  1. Выделите набор, к которому необходимо создать запрос.
Примечание: Если в теме находится только один набор данных, то при открытии Диспетчера базы данных он будет выбран автоматически, а справа отобразится инструмент фильтрации. Если имеется более одного набора данных, необходимо выделить требуемый набор данных, который будет открыт в окне фильтрации.
  1. Для тем, где данные содержатся в множественных структурах, необходимо в раскрывающемся меню выбрать структуру для построения запроса, например, объект, момент времени, интервал времени или статические свойства.


  1. После выбора атрибутов набора данных, по которому необходимо выполнить фильтрацию, перейдите к раскрывающимся меню инструмента фильтрации. Могут применяться различные фильтры в зависимости от доступных данных:

    Участки: Этот фильтр выделяет все записи, которые соответствуют как минимум одной из выделенных областей.

    Рабочие смены: Этот фильтр выделяет все записи, которые произошли как минимум в одной из выделенных смен.

    Метки: Этот фильтр выделяет все записи, которые содержат как минимум одну из выбранных меток.

    Фильтры свойств: Фильтрация данных при помощи фильтров свойств позволяет выбрать записи, соответствующие всем заданным фильтрам, объединенным при помощи стандартных операторов «есть», «не есть», >=, или <=. После настройки фильтров свойств, их можно включать или отключать, чтобы увидеть разницу между двумя наборами результатов. Фильтры свойств могут быть сохранены.

    Фильтр по дневному интервалу: Этот фильтр производит выборку всех записей в заданном количестве полных дней до сегодняшнего дня. Например, при запросе записей за «2 последних полных дня» будут выданы все записи за вчера и позавчера, независимо от времени суток и не включая сегодняшний день.

    Объект (имя фильтра изменяется в зависимости от выбранной темы): Список всех индивидуальных объектов в наборе данных. Эта функция позволяет выбрать объекты по имени (идентификатору). В зависимости от темы, имя фильтра объекта может быть следующим: пробы, скважины, пункты наблюдений, приборы для измерения напряжений, подвижные единицы оборудования или пункты наблюдений на стационарных установках.

    Имена групп: При фильтрации данных по именам групп выделяются все записи, которые содержат одно из выделенных имен групп.

    Материал пробы: Эта опция относится к пробам и предназначена для сортировки проб по заданному материалу, например газ, порода, грунт, вода и т.п.

    Значение находится ниже предела чувствительности: Данная функция позволяет выбрать способ отображения данных, которые находятся ниже предела чувствительности в табличном представлении (текстовый формат или числовое значение). Для построения диаграммы будет использоваться половинное значение предела чувствительности.


Примечание: Может быть задано более одного фильтра, в этом случае система выведет все записи, которые соответствуют всем фильтрам свойств.
  1. Нажмите Применить изменения на панели инструментов с правой стороны.



Примечание: Диаграммы и таблицы имеют стандартные настройки отображения, которые позволяют отправлять запрос немедленно (нажатием кнопки Применить изменения) без необходимости здания настроек диаграммы и таблицы. Вы можете внести изменения в настройки позже и повторить отправку запроса.
Подсказка: Внизу панели опций фильтров отображена таблица, в которой показаны статистические данные числовых свойств. Эта таблица удобна при выборе значений для сортировки данных при помощи фильтров свойств.

Параметры отображения диаграммы

Просмотр сортированных и несортированных данных может выполняться при помощи различных вариантов построения диаграмм, каждая из которых содержат множество свойств и настроек для точной регулировки данных и способа их представления. Кнопка «Диаграммы» находится справа на панели инструментов.

Примечание: Система может построить до максимум 50 000 точек на одной диаграмме.

Атрибуты диаграмм

Для построения каждого типа диаграмм необходимо задать свойства осей координат. Свойства могут быть заданы по выбору из списка.

Затем, в зависимости от типа диаграммы, становятся доступны дополнительные настройки, например группирование, радиус или цветовая модуляция, а также отображение подсказок.

Выбор типа диаграммы

По умолчанию, Geoscience INTEGRATOR выбирает первое свойство выбранного массива данных для отображения на оси x, и второе для отображения на оси y. В то же время, свойства для осей x и y могут быть заданы пользователем. Диаграммы типа «Гистограмма» содержат только параметры x и y, в то время как диаграммы типа «разброс», «временной разброс» и «троичное построение» позволяют добавление и просмотр дополнительных параметров.

Чтобы получить доступ к этим функциям, необходимо после построения диаграммы нажать кнопку Другое в меню Диаграммы. Дополнительные функции построения диаграмм:

Other chart properties Description
Радиус Настройка размера точек на диаграмме в зависимости от дополнительного числового свойства, помимо значений зна осях x и y.
Величина радиуса Настройка размера точек, где наивысшее значение соответствует наибольшему или наименьшему радиусу.
Цвет Назначение цветов точек в зависимости от числового свойства или классификации.
Цветовой спектр Назначение цветовой кодировки в зависимости от числовых свойств. Варианты: от красного (минимальное значение) до синего (максимальное значение) или от синего (минимальное значение) до красного (максимальное значение). При использовании свойства, заданного как классификация, присвоение цветов может быть выполнено в меню «Классификации».
Свойства подсказок Данная функция опрашивает отдельные точки на диаграмме и отображает одно или несколько свойств, связанных с каждой из этих точек. При выделении свойств и наведении курсора на отдельную точку появляется информационное окно, в котором отображаются значения свойства, соответствующие этой точке.

 

При нажатии кнопки Применить изменения на панели инструментов в правой области окна отправляется запрос, по которому строится диаграмма и таблица в соответствии с заданными фильтрами и параметрами отображения.

Параметры отображения таблиц

Пользователю проставляется возможность настраивать таблицу при помощи выделения и отмены выделений свойств в разделе Столбцы, сортировки данных по возрастанию и убыванию, а также выбора количества строк, которые будут отображаться на странице. Кнопка Таблицы находится справа на панели инструментов.

Созданная пользователем таблица отображается внизу диаграммы. В таблице содержатся только первые 300 строк результатов пользовательского запроса, но указывается полное количество найденных результатов. Данные могут быть дополнительно отсортированы нажатием на любой заголовок столбца, при этом сортировка выполняется по всему объёму результатов, выданных по запросу, а не только по первым 300 в списке.

При необходимости отображения специальных результатов, которые не входят в первые 300 строк, можно ограничить количество результатов запроса путем применения имеющихся инструментов построения фильтров и запросов.

Экспорт табличных данных (объём не ограничен 300 записями) в файл формата CSV выполняется нажатием на кнопку Экспорт в CSV сверху на правой стороне таблицы.

При нажатии кнопки Применить изменения на панели инструментов в правой области окна отправляется запрос, по которому строится диаграмма и таблица в соответствии с заданными фильтрами и параметрами отображения.

Добавление элемента отчёта

Созданные диаграммы и таблицы могут быть добавлены в отчёт как новый элемент. Чтобы добавить в отчет диаграмму или таблицу нажмите кнопку Добавить элемент отчета на панели справа, под кнопкой «Применить изменения». +

В диалоговом окне вам будет предложено выбрать отчет, к которому будет добавлен элемент, а также присвоить имя элементу.

Примечание: Фактически в отчете сохраняются параметры фильтров и отображения данных, а не результаты по текущему запросу. Таким образом, каждый раз при создании отчета снова будет выполняться запрос и будут выводиться наиболее свежие результаты. Элемент, добавляемый при помощи Диспетчера базы данных, состоит из диаграммы и таблицы.
