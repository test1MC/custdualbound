Библиотека документов

Среда Geoscience INTEGRATOR во многом отличается от традиционных геолого-геофизических баз данных. Одно из отличий заключается в способности системы сохранять файлы и документы и привязывать их к наборам данных, и даже к отдельным записям данных.

Поскольку загружаемые файлы могут быть разных типов, Geoscience INTEGRATOR предусматривает три разных способа для загрузки файлов в систему.

Документы, такие как отчеты, публикации, и презентации, могут быть загружены в библиотеку документов, где также хранятся связанные метаданные, например, год публикации и имена авторов.

Файлы проектов любого типа, независимо от того, содержится ли в них текст, лабораторные сертификаты, электронные таблицы, фотографии, трёхмерные триангулированные поверхности, блоковые модели и т.п., могут быть загружены в проект при помощи Диспетчера файлов.

Загрузка проектных чертежей, например, карт, планов горизонтов или разрезов выполняется при помощи страницы «Карты/планы/разрезы», где перечислены имена горизонтов для форматов DWG и DXF.

 

Документы

Среда Geoscience INTEGRATOR включает в себя систему управления документами, которая обеспечивает хранилище на уровне системы, в котором документы могут быть привязаны к отдельным проектам, наборам данных и меткам. Поиск документов выполняется по имени файла, по ссылкам на проекты или наборы данных или по их метаданным (автор, дата, название, и т.п.).

Поскольку документы, не привязанные к конкретным проектам, доступны для всех пользователей, библиотека документов может стать официальным информационным архивом вашего предприятия для хранения всех отчетов, презентаций, публикаций и справочных материалов.

  • При нажатии пункта меню Библиотека документов(i) открываются вкладки Мои документы (Загруженные вами документы) и Все документы (ii).
  • Кнопка Читать дальше (iii) открывает доступ к более детальной версии описания.
  • При нажатии указателем мыши на названии документа (iv) выполняется сохранение этого документа на жестком диске вашего компьютера.
  • При нажатии кнопки +Загрузить новый документ (v) выполняется переход на страницу Загрузить документы.

Дополнительная информация о библиотеке документов:

  • Новые документы могут быть добавлены любым пользователем.
  • Вы можете изменять или удалять только те документы, которые загрузили вы сами. (Эти функции доступны на вкладке Мои документы.)
  • Полномочия изменять или удалять любой документ на вкладке Все документы имеет только Администратор системы.
Примечание: Поскольку система разрешений доступа устанавливается для каждого проекта отдельно, просмотр документов, связанных с конкретным проектом, возможен только при наличии полномочий для доступа к этому проекту.

 

Загрузка документов

Модуль Загрузить документы содержит три вкладки:

  • Создать документ
  • Авторы
  • Связи

Новые документы

Для загрузки документов в библиотеку предусмотрена следующая серия обязательных и необязательных полей:

Field Description
Поле Название (Обязательное)
Год публикации (Обязательное)
Проект Позволяет ограничить доступ к документу, предоставив его только пользователям, которые имеют доступ к проекту. Если оставить поле незаполненным, доступ к документу будет открыт для всех пользователей.
Набор данных Данное поле появляется, если выделен проект, и отображает список имеющихся наборов данных для этого проекта. Документ может быть ассоциирован с множеством наборов данных.
Метки Данная опция появляется, если выделен проект, и отображает список имеющихся активных меток для этого проекта. Документ может быть ассоциирован со множеством меток.
Тип публикации (Обязательное)
Добавить автора Эта опция позволяет выделять, добавлять или править имена и связи начальных и дополнительных авторов. Для этого необходимо ввести первые буквы имени автора, и, после автоматического заполнения поля, выбрать роль и связи. Затем можно просмотреть информацию и, если дополнительные изменения не требуются, сохранить строку.

Авторы и связи

Система Geoscience INTEGRATOR составляет список авторов и связей. Вкладки Авторы и Связи позволяют добавлять и редактировать имена авторов и связи при помощи несложной серии форм. Эти списки доступны для выбора авторов и связей при создании или изменении документа.
