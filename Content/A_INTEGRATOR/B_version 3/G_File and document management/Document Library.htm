Overview

differs in many ways from traditional geoscience databases. One of these differences is its ability to store and link files and documents with data sets and even with individual data records.

As files can be of many types, offers three different ways to upload them in the system.

  1. Documents, such as reports, publications, and presentations, can be uploaded in the Document library where associated information like year of publication and authors names and affiliations are also stored.
  2. Project files of any type, whether they store text, lab certificates, spreadsheets, photos, 3D triangulated surfaces, block models, etc. can be uploaded within a project via the File manager.
  3. Project drawings like maps, level plans, or cross sections are uploaded via the Maps / plans / sections page, where layer names are listed for DWG and DXF formats.

Document library

contains a document management system that provides a system-level repository, where you can link documents to individual projects, to data sets and to tags. Documents are searchable by their file name, their links to projects or data sets, or by their various information fields (author, date, title, etc.).

Because documents that are free of any project association are available to all system users (regardless of project), the document library can become your company's official repository for all reports, presentations, publications, and references.

  1. Clicking on Document library in the menu presents the Documents page.
  2. The two tabs are My documents, for documents you have uploaded, and All documents.
  3. The Read more button allows you to see a more detailed version of the description.
  4. Clicking the title of the document downloads the document to your hard drive.
  5. Clicking the +Upload new document button navigates to the Upload documents page.

Access permissions

Document access permission is established per project, so you will be unable to view documents associated with a project that you do not have permission to see. Then the following applies:

  • Any user can add a new document.
  • You can only edit or delete the documents you uploaded yourself, from the My documents tab.
  • Only the System administrator can edit or delete any document from the All documents tab.

Uploading documents

The Upload documents page presents you with three tabs, detailed below:

  • New document
  • Authors
  • Affiliations

New documents

The following series of mandatory and optional fields are available to upload documents to the library:

Field Description
Title (Mandatory)
Year of publication (Mandatory)
Project Allows you to restrict access to the document to users with access to the project. Leaving the field empty gives access to all users.
Data set This option appears if you selected a project, and displays the list of existing data sets for the project. A document can be associated with multiple data sets.
Tags This option appears if you selected a project and displays the list of existing tags for the project. A document can be associated with multiple tags.
Publication type Pick from a preset list. (Mandatory)
Add author Allows you to select, add, or edit primary and additional authors’ names and affiliations. Do this by entering the first letters of an author's name, and when it populates, select the role and affiliation. You can preview it and save when you are satisfied.

Authors and Affiliations

builds a list of authors and affiliations. The Authors and Affiliations tabs allow you to add and edit authors and affiliations with a simple set of forms. These lists are available for Author and Affiliation selection at document creation or editing.
