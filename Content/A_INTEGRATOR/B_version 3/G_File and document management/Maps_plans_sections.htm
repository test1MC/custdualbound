Maps / plans / sections

This section is a repository within , much like the File manager, that allows you to upload, store, filter, and sort files related to maps, level plans, and cross sections of mining and exploration sites. The same list of File types used in the File manager is available here. You can associate Maps / plans / sections to Tags and Data sets, which is a very powerful tool as it allows you to rapidly identify elements that can go together even if they are of assorted formats.

The Maps / plans / sections page functions similarly to the File manager tool, with the additional feature of listing layer names from DWG and DXF file. It does not allow you to flag a file as a "data file". The layers list gives you information on the content of a map, plan, or section document without having to open it.
