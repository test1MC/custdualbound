File manager

The File manager is a repository within that allows you to upload, store, filter, and sort project-related files of different types. You can associate files to tags and data sets.

It is possible to associate a file to individual data records; however this is not done via the File manager, but in Reporting (see "Assigning tags and files at the record level" on page 1).

Managing existing files

  • Clicking on File manager in the menu opens the file filter tool and the files summary table.
  • The Filter files module allows you to filter by name, file type, date range, and user who uploaded.
  • Clicking a file name opens a dedicated page listing the data sets linked to the file.
  • The +Upload a new file button opens a module below the data summary table to upload a new file.
  • Clicking deletes the file, and clicking downloads the file to your hard drive. (Availability based on permission level.)
  • Clicking the number in the Tags column opens a dialogue box that lists the tags associated to the file, where you can assign and unassign tags to the file.
  • Clicking the number in the Data sets column opens a dialogue box that lists the data sets associated to the file, where you can assign and unassign data sets to the file.
  • Checking the box in the Data file column indicates that the file contains actual data that can be used to view in other applications.

Uploading new files

When you click the +Upload a new file button, you can use the form to associate file information, tags, and data sets with the file you are uploading.

Selecting tags and / or data sets from their respective lists will associate them to the file you are uploading. You can use the search box to quickly find items in the list. The tags and data sets in the list are those that are available in the project.

The list found in the file type drop-down menu can be managed in the Project settings (see "File types" on page 1).
