Filtering and querying data

The Data set explorer contains a powerful and flexible filtering tool to help you access subsets of data. There are many ways to customize the way you want to view and present your data through the numerous chart and table options.

You will find the filtering tool in the toolbar, as shown at right.

Once you have selected one of the tools (filters, charts, tables), you will see a series of drop-down menus that correspond directly to data defined within the data set. Therefore, the filtering options vary between data sets.

To filter data:

  1. Select the data set you want to query .
  1. For themes where data is in multiple structures, you must select the structure you want to query from the top drop-down menu, such as object, instant, interval, or static properties.
  1. Decide which attributes of the data set you want to filter, and navigate the drop-down menus available in the filtering tool. Many filtering options are available, depending on the available data:

Neighbourhoods: This filter will select all records that fall into at least one of the selected neighbourhoods.

Shifts: This filter will select the records that occurred in at least one of the selected shifts.

Tags: This filter will select all records that contain at least one of the selected tags.

Property filters: Filtering using property filters selects records that match all the defined filters, combined together, using the common operators is, is not, >=, or <=. When property filters are set, you can toggle on () and off () their application to view the difference between the two sets of results, as well as save them to quickly re-use them later from the Saved list.

In themes with multiple template categories (except for Samples), there is a check box to return all results for items that match the property filters. For example: querying the drillhole interval data and using the property filter on the property Au >= 6 g/t. Leaving the box unchecked will return only intervals that have Au value over 6 g/t. However, if you checked the box, the query would find all the drillholes that have at least one interval with Au value over 6 g/t, and return all the intervals for these drillholes, no matter what the Au value at each individual interval.

  1. Click Apply changes in the toolbar at right.

Charts and tables have default display settings that allow you to send the query right away (by clicking Apply changes ) without necessarily having to define chart and table settings. You can adjust the settings afterwards and apply the query again.

Chart display settings

You can view your query results through numerous chart options, each containing multiple attributes and settings to help you fine-tune the details of the data and how you wish to present it. You will find the Charts button in the toolbar, on the right.

The following chart types may be available, depending on the theme.

Scatter plot

The Scatter plot allows you to choose properties for both X and Y axes, as well as properties to scale the points' radius and colour by, and properties to display as tooltip on each point.

On scatter plots, you have the option to view the data on a logarithmic scale by toggling on the X-axis log scale and Y-axis log scale buttons.
The system will plot a maximum of 50,000 points on a single chart.

Ternary plot


The Ternary plot allows you to choose properties for three axes, as well as properties to scale the points' radius and colour by, and properties to display as tooltip on each point.

Remember that the Ternary plot does not report exact values but the relative proportion of all three variables (for each point, the sum of their three axes values is 100%).

Histogram

The Histogram chart allow you to see a distribution of your data with a few grouping options depending on the property you selected for the X axis, and a choice of sum / average / count of elements for the Y axis property.

Time scatter


The Time scatter works just like the Scatter plot (radius, colour, tooltip options), except that the X axis can only be a datetime property.

If a timestamp includes milliseconds, the value plotted on the chart is rounded up to the nearest second, whereas the table will display the precise value to the millisecond.

Time series

The Time series only allows datetime values on the X axis, and lets you select multiple properties on the Y axis, provided that they share the same unit. Each property will have its own coloured curve with a colour legend at the bottom.

If a timestamp includes milliseconds, the value plotted on the chart is rounded up to the nearest second, whereas the table will display the precise value to the millisecond.

Instant time series

When querying Instant data available on certain themes, you also have the Instant time series chart that works just like the Time series (X axis needs to be a datetime property and Y axis allows for multiple properties), but plots data per object. You can toggle objects on and off by clicking their names above the chart. You need to have selected at least one object from the filter panel.

If a timestamp includes milliseconds, the value plotted on the chart is rounded up to the nearest second, whereas the table will display the precise value to the millisecond.

Chart attributes

Each chart type requires that you define the properties to use for the axes. You can select properties of your choice in the list.

Then, depending on the chart type, additional settings are available like grouping, radius or colour modulation and tooltip display.

Choosing a chart type

By default, selects the first property of your data for the x axis, and the second for the y. However, you can select other properties for the x and y axes. Histogram charts only contain x and y axis parameters, while Scatter plot, Time scatter, and Ternary plots allow you to add and view additional dimensions.

To access these features, click the Other button in the Charts menu, once a chart is created. They are as follows:

Other chart properties Description
Radius Adjusts the size of the points on the chart based on a numeric property.
Radius value Adjusts the size of points where highest value equals highest or lowest radius.
Colour Assigns colours to the points based on either a numeric property or a classification.
Colour spectrum Assigns colours defined by either red (min) to blue (max), or blue (min) to red (max) for numeric properties. If using a property defined as a classification, you can define colours in the Classifications menu.
Tooltip properties Queries individual points within a chart, and returns one or several properties associated with that point. Selecting the properties of interest and hovering the cursor over an individual point displays an information box presenting the values of the property associated with the point.

Clicking the Apply changes button in the right-side toolbar will send the query and return a chart and a table as per defined filters and display setting.

Table display settings

You can customize a table by selecting properties from the list or deselecting properties by clicking their blue bubble in the Columns section, sorting data ascending or descending, and selecting the number of rows to display per page. You will find the Tables button in the toolbar, on the right.

The customized table will display below the chart. It will only contain the first 300 rows of your query but informs you of the total number of results. You can further sort by clicking any column header, which applies to the complete query results, not only the first 300.

If you are looking for specific results that are not in the 300 rows, you can constrain the query results by filtering further using the available filtering and querying tools.

You can export the complete table results (not limited to 300) in CSV format by clicking Export to CSV above the table, on the left.

Clicking the Apply changes button in the right-side toolbar will send the query and return a chart and a table as per defined filters and display setting.

Add report item

You can add any generated chart and table as a new item to a report by clicking the Add report item on the right panel.

You will be prompted to choose a report to add the item to, and to give the item a name.

You can add the item to multiple reports, as well as create a new report directly from the Add a new report item dialogue box. This saves you from having to exit the dialogue box and go through the creation process again in the user interface. To do so, type the new report name in the Create new report name text box and press + at right.

The report definition actually saves filters and display settings, not the current query results. This way, every time you generate the report, the query is re-run to return the latest results. The item added through the Data set explorer contains both the chart and the table.
