Data set search

The Data set search page is helpful if you are not familiar with the theme structure or with how the data was organized in the project. It searches the whole project and returns the corresponding data sets.

You can access it by clicking Data set search in the menu at left.

You can use the Name and Description boxes combined or separately to look up any data set with a matching string of characters in that field. The resulting data sets are grouped by theme. You can click any of them to see more details, and you can click the magnifying glass to navigate directly to the Data set explorer page in the right theme, ready to query their data, download their files or manage their metadata.
