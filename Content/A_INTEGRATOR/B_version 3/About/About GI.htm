About

is a 4D multi-disciplinary data management system for geotechnical, mining, and exploration data, models, and metadata. It allows the integration of data in a single, secure repository shared by all team members for retrieval, analysis, and decision making. This multi-user, multi-site data management system has powerful and intuitive query capabilities. It can generate and manage scheduled, data-triggered, or on-demand reports.

Its geohazard assessment module provides a framework for using all available data to better understand, forecast, and mitigate geohazards. It supports geotechnical real-time hazard assessment and delivers scheduled or triggered reports for real-time decision making.

As a true 4D data management system (x, y, z, time), handles 3D spatial and temporal data for any data type for which spatial position or time is relevant. The diagram below illustrates the general system configuration from a user perspective.


The system connects to site data sources for real-time, continuous update or ad hoc update. Data entry to the system is simple and can be automated according to user-customized specifications. Data streams are captured, quality-controlled, and conditioned by the system directly from web browser entry, from other site databases, or from flat files on a file network system. The management of documents, files, and their links to data is simple and effective.

The data management system is server-based, with client applications accessing the server over a private network or the internet. Users gain access through a fine-grained permissions system under the control of the system administrator. Two client applications are provided:

  1. A web browser for system setup, data management and data analysis; data import and export, data tagging, display of files, documents, tables, and charts of data; and customized report setup through either self-serve or system-defined reporting, accessed directly in the web browser or by automated email distribution.
  2. A powerful 3D visualization application with query interface, that displays data in the context of maps and 3D geological models.

Both the web browser and 3D visualization applications support complex spatial, temporal, and property-value queries. Additional applications can be developed by end users connecting to through a RESTful API (Application Program Interface).

Value

  • Provides a user-friendly system to query and analyze 4D data.
  • Facilitates knowledge transfer amongst mine personnel.
  • Focuses users on interpretation and analysis, not on data handling.
  • Adds value to systems already in place by increasing their efficiency and scope without removing any important outputs.
  • Ensures data integrity in a secure, managed environment.
  • Provides a stable archive of the mine’s geotechnical and production data which is captured for long-term archive, query, and retrieval.

Features

Analysis and reporting

  • Direct web browser access to data and reports.
  • Comprehensive chart and tabulation options for data analysis.
  • Innovative tags, shifts, and neighbourhood queries to personalize and interrogate data.
  • Custom-designed summary reports dispatched by email to designated recipients.
  • Scheduled, data-triggered, or on-demand reports.
  • Automated geotechnical hazard assessment computation, update, and reporting.
  • Powerful 3D visualization environment with a query interface that allows the selection of data by type, data set, time range, spatial location, value ranges, and other query parameters.

Integration and data management

  • Standardizes data structures with simple access and reporting rather than relying on spreadsheets.
  • Stores primary data, models, analysis results, metadata, files, and documents as well as maps and level plans in DXF / DWG and other formats.
  • Enables simple data entry that can be automated and customized.
  • Controls data quality with customized validation rules.
  • Connects to site data sources for real-time, continuous, or ad hoc update.
  • Captures, quality-controls, and conditions data streams from underground or other data sources: from web browser entry, from other site databases, or from flat files on a file network system.
  • Manages static and time-dependent data and models, along with their contextual information.
  • Automates capture and quality control of mine geotechnical data in an integrated system.

Admin IT

  • Multiple mines can be supported simultaneously with the same system.
  • Data can remain on site under client control or be remotely hosted.
