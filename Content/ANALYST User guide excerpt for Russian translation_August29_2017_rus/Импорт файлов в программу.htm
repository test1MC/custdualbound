Импорт файлов в программу Geoscience ANALYST

Существует два способа импорта файлов в программу Geoscience ANALYST: при помощи меню Импорт и перетаскивание мышью.

Импорт файла при помощи меню Импорт:

  1. Перейдите в меню Файл > Импорт.
  2. Выберите тип файла и объект, если необходимо.

Импорт файлов методом перетаскивания:

  1. Определите местонахождение файла в локальной системе.
  2. Перетащите файл в окно просмотра или на панель объектов. Программа Geoscience ANALYST определит тип расширения файла и запустит соответствующий фильтр импорта.

Примечание: Если окно просмотра используется на вспомогательном мониторе, перетаскивание файла возможно только в Панель объектов, а не в окно просмотра.

 Файлы AutoCAD

Вы можете создавать точки, кривые и поверхности путём импорта файлов .DWG и .DXF системы AutoCAD из обширного множества приложений, связанных с AutoCAD. После выполнения импорта программа Geoscience ANALYST сохраняет импортированные файлы в папке под названием AutoCAD imports в панели объектов.

 

 Файлы ASCII

Программа Geoscience ANALYST оснащена двумя средствами импорта CSV-файлов ASCII:

  1. Файлы, построенные на основе столбцов : Создание точечных объектов.
  2. Файлы скважин: Создание скважин из серии столбчатых файлов, включая файлы данных об устьях скважин и дополнительные файлы данных инклинометрии и прочих данных.

 Столбчатые файлы ASCII

При импорте файлов через меню или перетаскиванием открывается окно Точки импорта, в котором представляется возможность задать параметры данных так, чтобы программа Geoscience ANALYST точно воспроизвела данные, как показано на следующей таблице:

 

Опция

Описание

Имя

Имя точечного объекта, созданного в новой или существующей папке под названием Средства импорта ASCII.

Обязательные поля

Выбор координат Х/Y/Z (восточное указание по сетке координат, северное указание, отметки высоты). Средство импорта предпримет попытку автоматически обнаружить соответствующие поля по имени заголовка (например, X/Y/Z, восточное указание, северное указание, отметка высоты). Если средство импорта не обнаружит никаких полей в CSV-файле, тогда первые три столбца будут приняты как значения координат X, Y, и Z.

Дополнительные поля

Выбор значений NDV (значение без данных) и темы (эталонные данные или число с плавающей точкой) для каждого столбца в импортируемом CSV-файле. Служебная программа прогнозирует тему на основе предварительных значений первых 256 записей CSV-файла. Для текстовых записей программа устанавливает тип эталонных данных. Для числовых данных программа устанавливает тип данных как число с плавающей точкой. Программа не устанавливает оптимальный тип автоматически, правильный тип можно задать вручную при помощи раскрывающегося меню.

Глобальные параметры значений без данных

Функция ввода пользовательских значений без данных для отдельных столбцов. При вводе значений в глобальные параметры значений без данных, разделите данные запятыми и нажмите Применить. Значения будут немедленно добавлены в каждый столбец. Необходимо вводить значение полностью, как показано в окне предварительного просмотра данных, например, число «0.01» должно быть введено полностью, а не в сокращенном варианте, как «.01.».

Предварительный просмотр данных

Отображение первых 256 записей файла ASCII в режиме предварительного просмотра.

 ASCII-файлы скважин

Импорт ASCII-файлов скважин выполняется только при помощи меню импорта, поскольку в процессе импорта необходимы различные файлы и настройки параметров данных.

Перед загрузкой файла, в диалоговом окне Импорт файлов скважин пользователю предлагаются следующие опции:

 

Опция

Описание

Файл устья скважины

Просмотр локальных папок для выбора необходимого файла устья скважины.

Файл каротажных данных

Просмотр локальных папок для выбора необходимого файла скважинных исследований. Если оставить это поле незаполненным, программа будет рассматривать все скважины как вертикальные. Если программа не обнаруживает скважину в файле каротажных данных, но обнаруживает её в файле устья скважины, средство импорта создаст вертикальную скважину.

Файлы реестров интервалов

Выбор одного или более файлов данных, которые относятся к интервалам скважин. Все файлы должны быть в одной и той же папке.

Файлы реестров точек

Выбор одного или более файлов данных, которые относятся к точкам скважин. Все файлы должны быть в одной и той же папке.

 

Чтобы импортировать ASCII-файлы скважин:

  1. В окне Импорт файлов скважин нажмите Просмотреть и выделите устье скважины, данные измерений скважины и файлы данных на локальном диске.
  2. Нажмите OK.

 
  1. В окне Выделить требуемые столбцы, используя раскрывающиеся регуляторы, укажите, если необходимо перезадать Требуемые поля со столбцами, которые отличаются от предложенных служебной программой автоматического определения.
  2. Повторите шаг 3 для каждого типа файлов, поскольку они выводятся последовательно.

 Файлы изображений ГИС

Программа Geoscience ANALYST совместима с несколькими типами файлов изображений, при импорте каждого их которых создаётся объект ГИС. Сюда входят следующие типы файлов:

  • .jpg/.jpeg
  • .gif
  • .bmp
  • .tif/.tiff

Помимо этого, программа Geoscience ANALYST поддерживает два типа файлов геопозиционирования в формате ASCII:

  • Файлы World: горизонтально ориентированные изображения (карты в горизонтальной проекции)
  • Файлы местоположения формата .csv: горизонтально и вертикально ориентированные изображения (карты в горизонтальной проекции и разрезы).

Файл местоположения формата .csv состоит из трёх строк, определяющих координаты верхнего левого, верхнего правого и нижнего левого углов изображения, разделённых запятыми.

Формат файла World

Строка

Координаты

1

Компонента X ширины пикселя (шкала x)

2

Компонента Y ширины пикселя (смещение y)

3

Компонента X высоты пикселя (смещение x)

4

Компонента Y высоты пикселя (шкала y)

5

Координата X центра верхнего левого пикселя

6

Координата Y центра верхнего левого пикселя

Например, файл типа World, требуемый для импорта и геопозиционирования карты в горизонтальной проекции без вращения с ячейками размером 3,5х3,5 м с началом координат в точке 1900E, 55000N будет выглядеть следующим образом:

3,5
0,00
0,00
-3,5
1900
55000

Формат CSV-файла местонахождения

Строка

Координаты

1

координаты x, y, z верхнего левого угла

2

координаты x, y, z верхнего правого угла

3

координаты x, y, z нижнего левого угла

Так например, файл местонахождения для импорта и геопозиционирования вертикального разреза направления северо-восток от отметки высоты 3100м до отметки 2500м будет выглядеть следующим образом:

6600,16400,3100

6800,17010,3100

6600,16400,2500

Чтобы выполнить автоматическое геопозиционирование изображения после импорта, связанный файл World или файл местоположения должен находиться в той же папке, где находится изображение, и называться так, как и файл изображения. Файл world должен иметь такое же расширение, как и файл изображения, с добавлением «w» (например, jpgw), в то время, как файл местоположения должен иметь разрешение .csv, как файл, разделённый запятыми.

Примечание: Если в папке находятся одновременно файлы world и файлы местоположения, средство импорта будет использовать по умолчанию файл world. Если в папке не найдены файлы world или файлы местоположения, импорт изображения будет выполнен в виде горизонтального Изображения ГИС, с точкой начала координат 0, 0, 0 и размером каждого пикселя, соответствующим 1x1 единиц измерения.
