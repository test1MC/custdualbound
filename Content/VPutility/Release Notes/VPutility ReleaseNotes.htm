Overview

VPutility is an update to the free utility in VP Suite. The feature list in this release has expanded to better help you create and modify VP models.

New and updated functionality

  • Removal of constraints from a VP model file. Choose to remove any type of constraint or choose to remove all constraints were placed in an existing VP model file.
  • Transfer geometries of a VP model to another physical property. By adding a physical property definition to the geometry of an existing model, you now have a new physical property model that can be forward modelled or inverted with the other programs within VP Suite.
  • Ability to handle alphanumeric line data. Lines are no longer considered strictly numeric minimizing the need to preprocess input data files. Outputting the model to Geoscience ANALYST with lines will create a curve object, allowing you to go line-by-line in the profile viewer.
  • Input lateral heterogeneities when creating models. Physical properties can now optionally be added to the layer when creating a VP model or adding a layer in an existing VP model.
  • Create and output rotated models to Geoscience ANALYST. VPutility helps you creating rotated models for VPmg and output on a rotated mesh to Geoscience ANALYST. It can also rotate existing models for you if the model has had to be rotated in the past.


Output of a created rotated model at -20 degrees rotation from North.

  • Information mode. This useful mode shows (and prints to a log file) the number of cells, types of units and constraints, and the size of the derivative matrices (in GB) for inversion if data is present or given in the mode. It is done without an input file.
