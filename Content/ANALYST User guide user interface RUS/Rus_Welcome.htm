Введение

В этом документе представлен краткий обзор программного средства трёхмерной визуализации и коммуникации Geoscience ANALYST, разработанного компанией для создания и работы с комплексными, многопрофильными моделями геологической среды и наборами данных.

Geoscience ANALYST - это уникальное автономное приложение, позволяющее осуществлять импорт трёхмерных геологических и геофизических данных и моделей из стандартных отраслевых файловых форматов, а также изменять параметры отображения для выполнения контекстно-зависимых запросов данных, связанных с объектами моделей.

Программа Geoscience ANALYST доступна только в виде 64-разрядного приложения для операционной среды Windows компании Майкрософт. Приложение совместимо с системами Windows 7, 8, и 10.

 

Область применения

В этом документе даётся описание основных задач, позволяющих начать работу с проектом в программе Geoscience ANALYST, включая инструкции по следующим операциям:

  • Скачивание и установка программы Geoscience ANALYST.
  • Копирование лицензионной информации при запуске программы.
  • Загрузка имеющегося проекта.
  • Ознакомление с панелью объектов.
  • Изучение основ работы с панелью окна просмотра.

 

Скачивание и установка программы Geoscience ANALYST

Программное приложение Geoscience ANALYST является бесплатным продуктом, который доступен для скачивания на сайте компании Mira Geoscience.

Скачивание и установка программы Geoscience ANALYST:

  1. Перейдите по следующей ссылке.
  2. На Интернет-странице программы Geoscience ANALYST нажмите кнопку Download Now (Скачать).
  3. На портале загрузки Geoscience ANALYST отметьте пункт Geoscience ANALYST v 2_51_setup.exe и нажмите кнопку Download Selected (Выбор загрузки).
  4. Запустите скачанный файл и в окне мастера установки программы Geoscience ANALYST нажмите кнопку I accept the agreement (Я принимаю условия соглашения) (iii), затем нажмите кнопку Next (Далее).
  5. Повторно нажмите кнопку Next (Далее) или, по желанию, выберите другую папку для установки программы Geoscience ANALYST.
  6. Повторно нажмите кнопкуNext (Далее) затем, нажмите кнопку Install (Установить).
  7. По окончании процесса установки нажмите кнопку Finish (Готово).

 

 

Активация лицензии

При первом открытии программы Geoscience ANALYST будет запущен мастер лицензирования, который проведет вас через процесс активации лицензии.

Активация лицензии ПО Geoscience ANALYST:

  1. Запустите программу Geoscience ANALYST.
  2. В окне приветствия Welcome мастера лицензирования программа Geoscience ANALYST нажмите кнопку Register now for your free key (Регистрация для получения бесплатного ключа активации).
  3. В окне формы активации программы Geoscience ANALYST необходимо заполнить требуемые поля и нажать кнопку Send (Отправить).
  4. Проверьте вашу электронную почту, вам выслан код активации программы Geoscience ANALYST.
  5. В окне приветствия мастера лицензирования программы Geoscience ANALYST нажмите кнопкуNext Далее).
  6. В окне Activation Method (Метод активации) мастера лицензирования программы Geoscience ANALYST выберите Activate my license online (Активировать лицензию через Интернет) и нажмите кнопкуNext Далее).
  7. Введите код активации в полеActivation Key (Ключ активации) (v) и затем нажмите кнопку Activate (Активировать).
  8. После появления зеленого флажка подтверждения спешной активации нажмитеFinish (Готово).
