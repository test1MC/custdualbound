Пользовательский интерфейс программы Geoscience ANALYST

Пользовательский интерфейс программы Geoscience ANALYST состоит из нескольких панелей, каждая из которых выполняет отдельную роль в процессе управления и интерпретации данных рабочего проекта.

Строка меню

Содержит стандартные раскрывающееся меню, необходимые для взаимодействия пользователя с программой, а также дополнительные опции.

Область отображения системы координат

Обеспечивает обратную связь в реальном времени с отображением в системе координат XYZ, соответствующем перемещениям мыши, углу наклона камеры окна просмотра и параметрам направления наклона.

Таблица данных

Отображает различные уровни информации, относящейся к внутренней структуре выделенного объекта, а также структуру папки объекта. При выделении объекта в панели Объекты или непосредственно в Окне просмотра выполняется одна или несколько функций из следующего списка:

  • Выделение цветом данных, соответствующих данному объекту в таблице данных
  • Отображение уровня вкладки и непосредственной родительской папки объекта с мета-данными (обобщённая информация)
  • Отображение примечаний пользователя и прикреплённых файлов

В таблице данных встроены вкладки, соответствующие подчиненным уровням объекта, как например, узлы, треугольники, ячейки и журналы.

Вкладки содержат в себе несколько столбцов, состоящих из чисел с плавающей точкой, целых чисел и ссылочных данных. При нажатии на инструмент заливки вверху столбца, объект в окне просмотра, связанный с данными, выделенными при помощи заливки, приобретает более контрастный цвет, что позволяет получить более детальное топографическое представление объекта.

Выделение строк в таблице данных

Нажать указателем мыши на строке:

  • Выделение строки
  • Выделение соответствующей точки данных на объекте в окне просмотра

Нажать и удерживать клавишу Ctrl и указателем мыши выбрать несколько строк:

  • Выделение нескольких последовательно выбранных строк
  • Выделение соответствующих точек данных на объекте в окне просмотра

Нажать и удерживать клавишу Shift и указателем мыши выбрать сначала одну строку, затем еще одну строку:

  • Выделение двух выбранных строк и всех строк, расположенных между ними
  • Выделение соответствующих точек данных на объекте в окне просмотра

Сортировка данных в таблице данных

Для сортировки данных в восходящем и нисходящем порядке, нажмите на заголовок столбца. При каждом нажатии изменяется направление сортировки вверх или вниз; значок треугольника в заголовке указывает направление.

Экспорт значений таблицы данных

Предусмотрено два способа для сохранения данных из таблицы данных в файл формата CSV.

Чтобы экспортировать из таблицы данных объём данных, задаваемый пользователем, выполните следующие действия:

  1. Выделите необходимые строки
  2. Нажмите Ctrl+c, чтобы скопировать данные
  3. Откройте другую программу, например Excel, Блокнот и т.п.
  4. Нажмите Ctrl+v, чтобы вставить данные.

Чтобы экспортировать таблицу из вкладки в таблице данных полностью, щелкните слева от отображаемой таблицы и сохраните её необходимом месте.

objects panel

The objects panel occupies the top left portion of the Geoscience ANALYST user interface, listing all the objects, tools, and data available in your workspace.

Note: Highlighting an item in the name column will select the corresponding object in the viewport, and vice versa (both will be highlighted yellow.)

 

You can customize and organize your workspace by creating generic container folders within the existing folders that contain data related to your project. To do this, right click a parent folder and select Add group. You can then name the new folder and drag and drop it to the appropriate location within the object tree.

Add group

The object panel's file structure hierarchy is: Folder > group > object > value. Figure 2.8 demonstrates this structure and includes the icons associated with each level.

Viewport

The viewport panel is the 3D space where you view, select, and manipulate objects and models. It is the focal point of the software. The vertical toolbar located at the right of the panel provides numerous options and tools to control how you view the object in your workspace.

Object selection

Left-clicking an object in the viewport performs two functions:

  • Selection of the object
  • Selection of the object's associated data, in the Cells tab of the data table panel, located directly underneath the viewport.

Left-clicking while holding down Ctrl allows you to multi-select objects in the viewport.

Left clicking the viewport background will select the Workspace tab in the data table panel, located directly underneath the viewport.

Object navigation

The perspectives and angles available to view an object in the viewport are infinte. There are several ways to navigate an object with your mouse by clicking and dragging:

Rotate on X, Y, Z axes (left mouse button)
Pan (middle mouse button), Zoom (mouse wheel)
Zoom (right mouse button)

Geoscience INTEGRATOR query

Geoscience ANALYST is the 3D front end to connect to, query, and visualize data from Geoscience INTEGRATOR, the software solution by Mira Geoscience for 4D multi-disciplinary exploration and geotechnical data management system.

To connect to the server:

  1. In the Geoscience INTEGRATOR query window, click the Connect now button .
  2. In the credentials dialogue, enter your email, password, and the URL address of a valid Geoscience INTEGRATOR server.
  3. Click OK.
  4. In the drop-down menu of the Projects section of the credentials dialogue, locate the desired Geoscience INTEGRATOR project and click LOAD.

Once the project has loaded, the query tab will contain a series of parameters allowing you to perform queries on selected data themes and import into the workspace.

To run a query in a -connected workspace:

  1. Use the query parameters in the query tab to customize your data filter.
  2. Click Run.

 

Data Colours panel

Located at the bottom right of the interface, this panel provides options to associate an object's data with specific colour settings, either defined via a mapping function or set manually.

The display and available options in the Data Colours panel correspond with the selected and painted data in the viewport and data table.

The Data Colours table consists of two interfaces. The histogram view and graph view displays when the painted data is float or integer data. The reference data view displays when the painted data is reference data.

Histogram view: Float and integer data

To display the histogram view in the Data Colours panel, select one or both of the following:

  • An object in the viewport (or its corresponding list entry in the objects panel)
  • A column of float and integer data in the data table (click paint can tool.)

Data Colours menu

i. Toggle folder menu

This drop-down menu changes the folder level of the selected data. This feature is particularly useful for examining statistics on an individual drillhole versus an entire drillhole group. You can analyze all the drillholes that are in the parent group of the selected drillhole or all the drillholes in the workspace.

Toggle folder menu

ii. Statistics

This section displays the statistics of the selected data. The volume of samples displays to the right of the theme name along the bottom of the x-axis.


iii. Histogram and graph

This section displays the histogram data on the x and y axes. The x-axis represents the data range. There are two y-axes:

  • The left side represents the user-defined colour bar.
  • The right side represents the count.

The yellow right angle line indicates the point of intersection of the selected data value and the corresponding colour.

The red line represents the colour mapping function.

By default, the histogram contains 50 bins. You can display up to 500 bins as a customized view.


iv. Data filter slider

This slider controls the volume of data displayed. The data within the limits of the slider will correspond to how the selected object in the viewport displays. On the object in the viewport, the values below (left of) and above (right of) the slider bars will appear either grey or transparent, depending on the Hide no data/filtered values setting.

You can set the exact values to include and exclude with the value boxes at the left and right sides of the data filter bar.

Reset clip values: Click this button to remove the filtering and reset the data filter bar.

 

v. Colour table menu

This drop-down menu contains a vast selection of colour mapping transformations that allow you to customize the view of the selected object.

Invert colour table: Switches colours to the complementary colours of the selected colour table.
Flip colour table: Turns the colour table upside down, switching the low and high colours.

 

Tip: You can create custom colour tables by using an ASCII file with three columns, a fixed header, and the RGB values normalized to 256. Save the file in the directory where you have installed Geoscience ANALYST (e.g. C:\Program Files\Mira Geoscience\Geoscience ANALYST\tbl).

vi. Bin selection

This value box allows you to specify the number of bins displayed in the histogram.

2D Profile viewer

The 2D profile viewer allows easy visualization of multiple graphs of single or multi-channel (groups) numerical data (integer and floats) stored on curve objects. This panel allows you to:

  • Display 2D sections from a BlockModel along the profile lines, which is ideal for comparison of inverted geophysics results with actual line data.
  • View selections made in viewer simultaneously in the Viewport and vice versa with dynamic linking.
  • Implement various scaling and graph settings options to create custom views. All settings are saved with the workspace.

 

2D profile viewer
  • To maximize the profile viewing area, you can hide the settings section (left side of viewer) by sliding the vertical separator all the way to the left.
  • To view the lines between the first and last line (curve part index), move the the horizontal slider located at the top of the graph area left and right using the following methods:
    • Clicking and dragging the handlebar
    • Using the keyboard arrows
    • Manually changing the value on the Curve part index selector by typing in a new value or pressing the up and down arrows.

Clicking the green and red squares button will show and hide the start and end of line indicators in the viewport.

The profile viewing area displays the vertically stacked graphs, and if activated, a painted section view of data stored on a selected BlockModel will display underneath the graphs.

The X-axis represents the distance along the selected curve line. The option of plotting eastings and northings is available through the graph settings area explained below.
