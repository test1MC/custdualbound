Fast heterogeneous inversion

The speed at which VPem1D can handle a heterogeneous inversion is dependent upon the number of parameters and data decay curves. During normal heterogeneous inversion, the derivatives update as VPem1D solves for the cell conductivity beginning at the top of the prism and working its way down. The fast heterogeneous option (ILD = -10) solves the cell conductivities all at once. The global misfit will still attempt to reach 1.0, but the main difference is that the cell conductivities are solved for to represent the whole of the decay curve.

Using the heterogeneous example, the same single representative decay curve and the main model features are reproduced in approximately 15% of the time (65 seconds versus 438 seconds), although the global misfit was 4.3 (reduced from 397). This option is ideal if using VPem1D to recover a starting model for a full 3D inversion. It is useful for large data sets in which areas of interest need to quickly be targeted from conductivity. You could target those areas for further scrutiny using the general heterogeneous inversion without waiting for the entire survey to invert.

Using multi-moment transmitters

Current EM systems sometimes elect to have more than one transmitter moment in flight, such as heliTEM or SkyTEM. VPem1D can forward model and invert two different transmitters. However, a few caveats come with this use:

  • The observed data must all be in the same file.
  • The observed data must all be of the same type (e.g., dBz/dt).
  • The given global percentages and floors are used throughout.
  • The data channel centres and widths are given for each transmitter.
  • The transmitter moments, waveforms, frequency, and pulse may differ.

To use this option, make sure that the parameter file begins with the header
#VPEM_MULTI#. At the end of the parameter file, the second transmitter information will begin using the #TXMOMENT# header. What follows will be the same as the last few entries of the single transmitter parameter file:

  • The data centre and widths section (as with the first transmitter).
  • The line with transmitter frequency, moment, and pulse.
  • The ramp-on and ramp-off parameters.
  • The optional waveform line.
