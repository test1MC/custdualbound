Overview

The release of VPem1D introduces a new feature: a best-fitting half-space calculation. Some minor bug fixes were addressed, and other features were updated to enhance usability. Additionally, the newly updated VPutility 1.1 has also been added to the installer.

New and updated functionality

We have performed bug fixes (including two notable ones) and introduced a new feature in VPem1D . Below is a list of upgrades to VPem1D:

  • Half-space optimization. A best-fitting half-space calculation can be performed prior to an inversion or set as the inversion style itself. Checks for all-zero models will also perform this optimization.
  • Alphanumeric line names. Line names in the Geosoft XYZ (.xyz) and CSV files may now contain letters as well as numbers.
  • Control of simplifying the waveform. You now have more control on the quantity of waveform simplification in the parameter file. This optional feature is added prior to the waveform file name.
  • Fixed conversion bug with GeoTEM dimensional dB/dt data. We found an internal conversion bug when dealing with data in units of pV/m2 and have fixed it. The conversion chart remains the same as in previous versions. This bug did not affect data in ppm.
  • Removed limit of number of geologic units. You are now allowed to have more than 99 geologic units.
  • Fixed bug in z-column of output statistics files (.csv). The easting and northing of the prisms were correct and now the elevation is too.
  • Check for zero conductivity. VPem1D now checks for zero conductivity and will give an error if units are found with conductivity less than 1e-7 mS/m.
