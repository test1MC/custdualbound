Parameter file

A parameter file (.par) file defines the name and format of the data file, and specifies the data channels selected for inversion. For multi-channel potential fields inversion, you must specify a PAR file in the control file. For single channel inversion, specify either a PAR file or a data file.

#VPMG#  
data file name  
nvar  
nch  
column1 chid1
column2 chid2
...  
columnnch chidnch
#VPMG# Required header.
nvar Number of fields (columns) in the data file. The minimum value is 4: east, north, elevation, and reading.
nch Number of data channels (or components) for modelling and inversion.
columnn Column number for the nth data channel.
chidn

3(or 4)-character ID for the nth data channel. The legal channel identifiers are:

  • FTG gradiometry: GXX, GXY, GYX, GXZ, GZX, GYY, GYZ, GZY, GZZ
  • VK1 gradiometry: GGA & GGC for along-line and cross-line
  • Falcon gradiometry: GUV & GNE
  • Total magnetic intensity: TMI or RMI
  • Magnetic gradients: MXX, MXY, MXZ, MYY, MYZ, MZZ, MTG or TMG (Total Magnetic Gradient)
  • Magnetic components: BX or BX_, BY or BY_, and BZ or BZ_
  • Magnetic amplitude data: AMP or MAP
  • Gravity data: GZ or GZ_ identifier
  • Per datum uncertainties: add an 'E' to the end of any component list (i.e. TMIE or GXYE). Overrides the control file unless not specified (multi-component).
  • Data-based rotations: DEC (per datum rotation to override the control file)
  • Specification of surface / drillhole data: IDH Overrides control file to check for prisms above topographic surface
  • Line values (can be numeric or alphanumeric): LINE

The values from the control file that can be given within the par file are the rotations (DEC identifier) and specification of surface or drillhole data (IDH). The values of these two identifiers follow the same rules as the control file.

Data file

This file contains the observed data. The data file may be a Geosoft XYZ file (.xyz) with "Line" or "Tie" alphanumeric line information. The first three columns must be easting, northing, and elevation. Dummy values for data are denoted as -99999.

east1 north1 elev1 obs11 obs12 ... obs1m
east2 north2 elev2 obs21 obs22 ... obs2m
...            
eastn northn elevn obsn1 obsn2 ... obsnm
eastn Easting (m) of measurement point at location n.
northn Northing (m) of measurement point at location n.
elevn Elevation of measurement point at location n (m).
obsnm

mth component for the nth location. Data units are as follows:

  • Free-air gravity: mgal
  • Gravity gradient: Eotvos
  • TMI: nT
  • Magnetic component: nT/m
  • Magnetic amplitude data: nT
