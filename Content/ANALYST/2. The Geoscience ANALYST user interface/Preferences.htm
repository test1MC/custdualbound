Preferences

By going to Edit / Preferences, you can open the dialogue box where you control the following elements of the workspace:

General

Parameter Description
User name Text field where you can enter and modify your user name.
Languages Drop-down menu to toggle between languages.
Camera rotation Drop-down menu to toggle between the default or free-form use of the camera in the Viewport.

Shininess settings

Parameter Description
Enable Check box to toggle on and off the radiance / shine of objects in the Viewport.
Surface / Point / Drillhole
shininess / intensity		 Use the spin box or slider to adjust the shine and intensity of the corresponding object in the Viewport.
